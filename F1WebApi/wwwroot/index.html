Introduction

This documentation will guide you through the usage of the F1 Web API. It provides information on available endpoints, how to interact with them, and includes code snippets and usage examples for the API.

Endpoints

Information on available endpoints and how to interact with them.

Endpoints Drivers:

Get all drivers

GET /api/drivers

Get driver by Id

GET /api/drivers/2

Get driver by name

GET /api/drivers?name=lewis

Examples

GET: localhost:port/api/drivers 

{
        "id": 1,
        "name": "Lewis Hamilton",
        "team": "Mercedes",
        "country": "United Kingdom",
        "Image": "lewis-hamilton.png"
},
{
        "id": 2,
        "name": "Valtteri Bottas",
        "team": "Mercedes",
        "country": "Finland",
        "Image": "valtteri-bottas.png"
}
.....
GET by Id: localhost:port/api/drivers/4 

{
       "id": 4,
       "name": "Fernando Alonso",
       "team": "Aston Martin",
       "country": "Spain",
       "Image": "fernando-alonso.png"
}

Endpoints Teams:

Get all teams

GET /api/teams

Get team by Id

GET /api/teams/2

Get team by name

GET /api/teams?fullTeamName=ferrari

Examples

GET: localhost:port/api/teams
